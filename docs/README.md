## Соглашения по документации

- **[Решение]** — проектное решение, принятое по умолчанию, требует явного подтверждения
- **[Гипотеза]** — предположение, которое нужно проверить до реализации
- **[Требует данных]** — раздел нельзя финализировать без данных с сайта или от заказчика
- **[Ядро]** — обязательная часть продукта
- **[Улучшение]** — желательно, но не блокирует запуск

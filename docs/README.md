## Маркеры в документах

- `[Решение]` — принятое проектное решение
- `[Гипотеза]` — требует проверки
- `[Требует данных]` — нельзя финализировать без внешних данных
- `[Ядро]` — обязательная часть
- `[Улучшение]` — желательно, не блокирует запуск
- `[СЛУЖЕБНЫЙ]` — используется AI-системой напрямую

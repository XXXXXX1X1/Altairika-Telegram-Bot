# Plan → Do → Verify

Любая задача = три шага. Не пропускай.

**1. ПЛАН** — перед тем как делать:
- Разберись в задаче (читай код, контекст)
- Определи критерии проверки: как убедишься что сделано правильно?
- Крупная задача → уточняющие вопросы + план пользователю

**2. ВЫПОЛНЕНИЕ** — делай по плану:
- Следуй плану, не уходи в сторону
- Проблема в процессе → вернись к шагу 1

**3. ПРОВЕРКА** — перед тем как сказать "готово":
- Проверь по критериям из шага 1
- Перечисли что проверил: `"Проверил: X ✓, Y ✓, Z ✓"`
- Нет проверки = не готово

**Фейл проверки:** диагноз → скорректируй план → повтори. Макс 3 итерации, потом — к пользователю.

**Масштаб:**
- Крупная задача → полный PEV + план пользователю
- Средняя → план в голове, проверка обязательна
- Мелкая → только проверка

Сомневаешься → выбирай более тщательный режим.
